Wanderlust 🗺️
Wanderlust is a full-stack web application inspired by popular vacation rental websites. It allows users to browse, create, and review property listings from around the world. The application features an interactive map for each listing, user authentication for secure access, and a responsive design for a seamless experience on any device.

🔗 Live Demo
[https://wanderlust-travel-xg3u.onrender.com/listings]

✨ Features
Browse & Discover: View a wide range of property listings with detailed descriptions, images, and pricing.

Interactive Maps: Each listing displays its location on an interactive map using the Mapbox Geocoding API.

User Authentication: Secure user registration and login system with session management.

CRUD Functionality: Authenticated users can create, read, update, and delete their own listings.

Reviews & Ratings: Users can leave reviews and ratings on listings to share their experiences.

Image Uploads: Listings include image uploads handled via Cloudinary for cloud-based storage.

Responsive Design: The user interface is fully responsive and built with Bootstrap for a great experience on desktops, tablets, and mobile devices.

🛠️ Tech Stack
Backend: Node.js, Express.js

Database: MongoDB with Mongoose

Frontend: EJS (Embedded JavaScript), HTML5, CSS3, Bootstrap

APIs & Services:

Mapbox API: For geocoding and displaying interactive maps.

Cloudinary API: For cloud-based image storage and management.

Authentication: Passport.js for session-based authentication.
